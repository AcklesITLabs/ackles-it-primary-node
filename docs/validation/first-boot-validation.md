# First Boot Validation

## BIOS Checks

- [ ] CPU detected correctly
- [ ] RAM detected correctly
- [ ] RAM installed in A2/B2
- [ ] EXPO enabled
- [ ] Storage drives detected
- [ ] CPU temperature normal at idle
- [ ] Fan curves visible
- [ ] Boot drive selected

## Hardware Checks

- [ ] GPU detected
- [ ] 12VHPWR cable fully seated
- [ ] Both EPS CPU cables connected
- [ ] AIO pump running
- [ ] All fans spinning
- [ ] No unusual noise
- [ ] No burning smell
- [ ] No warning LEDs remain lit
