# Beginner-Safe Build Checklist

## Stage 1 – Preparation

- [ ] Clear workspace
- [ ] Unbox parts
- [ ] Inspect components for damage
- [ ] Organize screws and cables
- [ ] Review motherboard manual
- [ ] Prepare anti-static precautions

## Stage 2 – Motherboard Preparation

- [ ] Install CPU
- [ ] Install RAM in A2/B2
- [ ] Install M.2 drives
- [ ] Verify motherboard standoffs in case

## Stage 3 – Case and Power

- [ ] Install motherboard
- [ ] Install PSU
- [ ] Route 24-pin ATX cable
- [ ] Route both EPS CPU cables
- [ ] Route GPU power cable

## Stage 4 – Cooling

- [ ] Mount AIO
- [ ] Install bottom intake fans
- [ ] Install top exhaust fans
- [ ] Install rear exhaust fan
- [ ] Connect fans to hub

## Stage 5 – First Boot

- [ ] Connect display, keyboard, mouse
- [ ] Power on
- [ ] Enter BIOS
- [ ] Confirm CPU, RAM, and drives are detected
- [ ] Enable EXPO
- [ ] Set boot order
