# Ackles IT Labs – Primary Node Deployment

## Overview

This repository documents the planning, architecture, build preparation, and validation process for the Ackles IT Labs Primary Node.

The system is designed as a high-performance workstation and lab platform for cybersecurity study, virtualization, infrastructure documentation, and future compute workloads.

## Current Status

**Status:** Build planning complete  
**Version:** v1.0 locked configuration  
**Phase:** Procurement and physical build preparation  

---

## System Roles

This system is intended to serve as:

- A primary workstation  
- A cybersecurity lab platform  
- A virtualization host  
- A high-performance compute node  
- A portfolio-grade infrastructure project  

---

## Project Goals

- Design a high-performance AM5 workstation  
- Document component decisions and tradeoffs  
- Validate compatibility, power, cooling, and expansion  
- Build a repeatable deployment record  
- Use the project as professional portfolio evidence
